TIL220525

Author: Goat95 Date: May 25, 2022  20:29:34 Category: Today I Learned

  • browser를 통해 페이지를 보는 과정에 대해 배웠다.
    • 주소창에 url 입력
    • 해당 주소로 http get request
    • 해당 주소에 있는 서버가 response로 html file을 보내줌
    • 브라우저가 해당 response를 화면에 render
  • Single Page Application에 대해 배웠다.
    • 자연스러운 페이지 이동 UX 제공 가능
    • 새로고침 발생x
    • render될 dom을 교체하는 작업이 필요하다는 단점이 있다.
