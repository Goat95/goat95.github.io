TIL220509

Author: Goat95 Date: May 9, 2022  16:49:51 Category: Today I Learned

오늘은 저번 주에 시작한 토이 프로젝트를 계속 진행했다.

메인 페이지의 마크업과 스타일은 마무리했지만, JS를 사용한 동적인 요소를 추가해야 했다.

Swiper JS를 사용하여 슬라이드 기능을 추가하고, 데이터가 저장된 json파일을 fetch 함수를 통해 연결하여 데이터를 가져오는 기능을 추가했다.
내일은 받아온 데이터를 리스트로 뿌려주는 기능과 페이지 이동 기능 및 버튼 클릭 기능을 추가해야 한다.

HTML 파일

  • Swiper JS를 사용하기 위해 head태그 안에 스크립트 파일과 CSS 파일을 연결하고,
  • swiper 클래스 안에 swiper-wrapper 클래스 안에 swiper-slide 클래스를 넣어주면 마크업은 끝나고, JS로 swiper 클래스 선택자를 불러서 실행하면 됩니다.

JS 파일

main.js

fetch.js

  • 처음에 json 파일에서 데이터를 받아오지 못해서 당황했는데 찾아보니 parcel-bundler를 사용했기 때문에 정적 파일을 연결하기 위해서 parcel-plugin-static-files-copy 패키지를 설치하고,
  • package.json에 “staticFiles”: {“staticPath”: “static”} 코드를 추가해주고, static 폴더를 만들어 그 안에 정적 파일을 넣어야 실행이 된다는 것을 알았습니다.
